# Holodeck
Unity Core for the Holodeck Usecase of [DroneManager](https://github.com/Domattee/DroneManager). Read the [docs](https://dronemanager.readthedocs.io/en/latest/holodeck.html) on how to use it.

The full project assets (under 2GB) are hosted as a formal release to ensure the main repository remains lightweight.

### How to Download
1. Navigate to the **[Releases](https://github.com/bakethi/Holodeck/releases)** section.
2. Locate the **Assets** section at the bottom of the latest release.
3. Download the `Holodeck.7z`.

### Installation
1. Once downloaded, extract the ZIP file to your desired location.
2. Follow the [docs](https://dronemanager.readthedocs.io/en/latest/holodeck.html) on how to use it.
